Maintenance Prédictive

NASA CMAPSS — Prédiction de RUL

Prédire la durée de vie résiduelle de moteurs industriels à partir de signaux capteurs.

Python pandas scikit-learn Random Forest Feature Engineering SQLite Streamlit Plotly

Projet Data Science de bout en bout sur le dataset NASA CMAPSS FD001. L'objectif est de prédire le Remaining Useful Life (RUL) d'un moteur industriel à partir de signaux capteurs, afin d'anticiper les risques de défaillance et de prioriser les interventions de maintenance.

  • Ingestion et structuration des données NASA CMAPSS FD001
  • EDA des 21 capteurs — suppression des 13 capteurs quasi-constants (std < 0.5)
  • Feature engineering : rolling mean 5 cycles, delta cycle à cycle — 24 features retenues
  • Cap du RUL à 150 cycles (aligné sur la plage du test set NASA)
  • Modélisation : Ridge baseline → Random Forest Regressor
  • Évaluation : RMSE, MAE, NASA Score (métrique asymétrique PHM'08)
  • Classification des moteurs : Critique (≤30 cycles) · À surveiller (≤60) · Stable (>60)
  • Dashboard Streamlit : vue opérationnelle, table de priorisation, analyse moteur
  • Couche analytique SQLite — requêtes métier réutilisables

Détection d'anomalies · Aviation

Aviation Risk — Détection d'anomalies de vol

Score de risque explicable sur données de trafic aérien réel — OpenSky Network.

Python OpenSky API pandas Isolation Forest Risk Scoring Streamlit Plotly

Projet construit autour de données de vol temps réel issues de l'API OpenSky Network. L'objectif est d'analyser des signaux de vol — vitesse, altitude, taux vertical — pour produire un score de risque explicable de 0 à 100, basé sur des contraintes physiques du domaine aérien (enveloppes ICAO), puis de le valider par un benchmark Isolation Forest.

  • Récupération de données de vol via l'API OpenSky Network (temps réel)
  • Nettoyage : suppression des vols au sol, gestion des nulls, typage
  • Feature engineering : speed_kmh, altitude_diff, vertical_rate_abs, lat/lon bins
  • Score de risque explicable 0–100 basé sur 5 règles physiques pondérées (ICAO)
    • Vitesse anormale (<100 ou >1 200 km/h) — poids 25%
    • Altitude anormale (<0m ou >13 000m) — poids 25%
    • Taux vertical extrême (>50 m/s) — poids 20%
    • Écart baro/géo élevé — poids 15%
    • Variation verticale forte — poids 15%
  • Classification : LOW · MEDIUM · HIGH + justification lisible par vol
  • Benchmark Isolation Forest — entraîné sur signaux bruts, indépendant des règles
  • Dashboard interactif Streamlit : filtres, carte, distribution des risques
Résultat clé
98,1 % d'accord entre le score rule-based et l'Isolation Forest sur 9 503 vols analysés. Les 1,9 % de divergences constituent les cas les plus intéressants à investiguer.
